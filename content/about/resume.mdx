# 안녕하세요 <br /> 프론트엔드 개발자 조효형 입니다. 😀

<!-- **저는 아래에 `관심`있어 하고 `고민`합니다.** -->

- UI/UX에 관심이 많고
- 기술보다는 사용자를 더 중요하게 생각하고,
- 몰랐던 것을 알고 새로운 것을 배우는 것을 좋아합니다.

## 기술역량

- react를 사용한 CSR 기반 프로젝트에 익숙합니다.
- typescript를 활용하여 타입 추론하여 개발하는것에 익숙합니다.
- javascript 문법에 능숙하며, 꾸준히 학습중 입니다.
- css는 scss를 사용하는것에 가장 익숙합니다.

## 경력

### mobidoo(sauce-live)

|          |                                                       |
| :------: | ----------------------------------------------------- |
| **기간** | 21.10 ~ 22.03                                         |
| **스택** | react, typescript, redux, redux-saga, storybook, sass |
| **작업** | 플레이어 리팩토링                                     |

#### 플레이어 리팩토링

- 현재 맡고있는 프로젝트중 소스플렉스 플레이어 관련 리팩토링 작업입니다.

#### 작업 내역

- 뷰와 로직들이 얽혀있는 컴포넌트들을 구분하면서 로직개선 및 코드정리
- styled-coponents를 걷어내고 module-scss적용하여 초기로딩속도 개선 및 번들사이즈 축소
- Icon이미지를 스프라이트 이미지로 적용하여 불필요한 이미지 요청 횟수 감소
- .env 적용하여 파일단위로 환경변수 관리
